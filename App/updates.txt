=========================================================================================================
LOADING FLOW
=========================================================================================================

CSV Uploaded
      │
      ▼
Parse CSV
      │
      ▼
Display Dummy Charts
("Dummy Data - Checking Cache...")
      │
      ▼
Search Cache
      │
      ├──────── Cache Hits Found ────────► Update UI Immediately
      │                                      (Real Cache Data)
      │
      ▼
Send Remaining Transactions to LLM
      │
      ▼
LLM Returns Categories
      │
      ▼
Update UI Again
(Final Categorisation)

If NO cache exists:

Dummy Charts
      │
      ▼
LLM
      │
      ▼
Final Charts
